Para visualizar el proyecto debes ir a este link https://github.com/duvanzm/PruevaM3Javascript

Pasos:

1. Debes clonar O descargar El archivo
2. Busca el archivo descargado o si vas a clonar desde git ve a la terminal
3. Desde la terminal usa este comando: git clone https://github.com/duvanzm/PruevaM3Javascript
4. Luego usa este comando: cd PruvaM3Javascript
5. Luego usa este comando: code .
6. cuando hallar abierto el proyecto en vsc
7. ejecuta este comando desde la terminar de vsc: npx json-server db.json
8. Busca el archivo index.html y con click derecho abrelo en el navegador.
9. Ya puedes visualizar el proyeto
